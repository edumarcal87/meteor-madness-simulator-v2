Como funciona

Fontes de dados: usamos NASA NEO (tamanho / velocidade) e contexto de terreno inspirado em dados do USGS (oceano, costa, interior, floresta) para efeitos educativos.

  • Bandas: Tamanho (pequeno/médio/gigante) e Velocidade (lento/rápido/super) são faixas simplificadas.
  • Índice de energia: energy_index = peso(tamanho) × peso(velocidade), variando de 1 a 9.
  • Níveis: Para cada terreno, comparamos o índice com os thresholds para classificar: BAIXO, MÉDIO ou ALTO.
  • Observação: É um modelo educativo, não uma simulação física completa.

Use o botão “🔭 Usar Exemplo da NASA” para preencher com um objeto real (simplificado) e ver o resultado.
